此 App 需部署到網路才能讓其他人開啟。請依下方步驟免費部署到 GitHub Pages：
部署步驟（3分鐘完成）：
1. 前往 github.com 登入（或註冊免費帳號）
2. 點右上角 ＋ → New repository，命名如 japan-trip
3. 進入 repo → Add file → 上傳此 HTML 檔，重命名為 index.html
4. 進入 Settings → Pages → Source 選 main branch → Save
5. 約1分鐘後網址 你的帳號.github.io/japan-trip 即可使用！
